# AgroSmart 🌱

AgroSmart is a smart agriculture web application built using Python, Machine Learning, and Deep Learning. It helps farmers and users make better decisions by providing crop recommendations, fertilizer suggestions, plant disease detection, and chatbot-based support.

---

## 🚀 Features

* 🌱 Crop Recommendation using Machine Learning (Random Forest)
* 💊 Fertilizer Suggestion System (Rule-based logic)
* 🌿 Plant Disease Detection using Deep Learning (ResNet9 - PyTorch)
* 💬 Smart Chatbot (TF-IDF + Logistic Regression)
* 🌦️ Weather Integration using OpenWeather API
* 📊 Efficient Data Handling using Pandas & NumPy

---

## 🛠️ Tech Stack

* Python
* Flask (Backend)
* Machine Learning (Scikit-learn)
* Deep Learning (PyTorch, ResNet9)
* HTML, CSS, JavaScript (Frontend)
* Pandas, NumPy
* Pickle (Model Serialization)
* OpenWeather API

---

## ⚙️ How It Works

1. User inputs farming-related data
2. Flask backend processes the request
3. ML model predicts crop recommendations
4. Deep Learning model detects plant disease from images
5. Chatbot responds to user queries
6. Weather API provides real-time environmental data

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/Anuj-2770/AgroSmart.git
```

2. Navigate to project folder

```
cd AgroSmart
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open in browser

```
http://127.0.0.1:5000/
```
